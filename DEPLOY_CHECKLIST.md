# ✅ Vercel Deployment Checklist

## 🎯 Quick Deployment Guide

Follow these steps in order:

---

## 1️⃣ Vercel Account Setup

- [ ] Go to: https://vercel.com/signup
- [ ] Click "Continue with GitHub"
- [ ] Authorize Vercel
- [ ] Complete signup

**Time:** 1 minute

---

## 2️⃣ Import Repository

- [ ] Go to: https://vercel.com/new
- [ ] Find repository: `Samraatsharma/eventra`
- [ ] Click "Import"
- [ ] Verify Next.js is detected

**Time:** 30 seconds

---

## 3️⃣ Add Environment Variables

Add these 6 variables (copy from VERCEL_QUICK_DEPLOY.md):

- [ ] `NEXT_PUBLIC_FIREBASE_API_KEY`
- [ ] `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN`
- [ ] `NEXT_PUBLIC_FIREBASE_PROJECT_ID`
- [ ] `NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET`
- [ ] `NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID`
- [ ] `NEXT_PUBLIC_FIREBASE_APP_ID`

**Important:** Select "All" environments for each!

**Time:** 2 minutes

---

## 4️⃣ Deploy

- [ ] Review settings
- [ ] Click "Deploy" button
- [ ] Wait for build to complete
- [ ] See success message

**Time:** 2-3 minutes

---

## 5️⃣ Test Deployment

- [ ] Click "Visit" button
- [ ] Home page loads correctly
- [ ] Explore page works
- [ ] Login page displays
- [ ] Booking page works
- [ ] Dashboard accessible
- [ ] Mobile responsive
- [ ] No console errors

**Time:** 2 minutes

---

## 6️⃣ Configure Firebase

- [ ] Go to: https://console.firebase.google.com
- [ ] Select project: `eventra-5443d`
- [ ] Go to: Authentication > Settings
- [ ] Scroll to: Authorized domains
- [ ] Click "Add domain"
- [ ] Add your Vercel URL
- [ ] Save changes

**Time:** 1 minute

---

## 7️⃣ Seed Database

- [ ] Visit: `https://your-vercel-url.vercel.app/admin/seed`
- [ ] Click "Check Database" button
- [ ] Click "Seed Database" button
- [ ] Wait for success message
- [ ] Verify 8 vendors added

**Time:** 1 minute

---

## 8️⃣ Final Testing

- [ ] Visit home page - see featured vendors
- [ ] Click "Explore Services" - see all vendors
- [ ] Test filters (category, location, rating)
- [ ] Click on a vendor - see profile
- [ ] Test signup - create account
- [ ] Test login - sign in
- [ ] Visit dashboard - see metrics
- [ ] Test booking form
- [ ] Check mobile view
- [ ] Test all animations

**Time:** 5 minutes

---

## 9️⃣ Update Documentation

- [ ] Copy your Vercel URL
- [ ] Update README.md with live demo link
- [ ] Commit and push changes
- [ ] Verify auto-deployment works

**Time:** 2 minutes

## ⚠️ Common Issues

### Build Failed?
**Solution:** Check environment variables are correct

### Firebase Not Working?
**Solution:** Add Vercel domain to Firebase authorized domains

### 404 Errors?
**Solution:** Verify all pages exist and routes are correct

### Slow Loading?
**Solution:** Check if database is seeded
